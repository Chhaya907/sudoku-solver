# sudoku-solver
sudoku solving project

 A simple and efficient Sudoku Solver built using Java.
This project uses the Backtracking Algorithm to solve a 9×9 Sudoku puzzle.

 # Features

1 Solves standard 9x9 Sudoku puzzles

2 Uses Backtracking algorithm

3 Clean and simple console-based interface

4 Beginner-friendly implementation

5 Well-structured and readable code

# Algorithm Used

1 This project uses the Backtracking algorithm, which works by:

2 Finding an empty cell

3 Trying numbers from 1–9

4 Checking if the number is valid:

5 Not in the same row

6 Not in the same column

Not in the same 3x3 subgrid

Recursively solving the rest of the board

Backtracking if a number leads to a dead end
